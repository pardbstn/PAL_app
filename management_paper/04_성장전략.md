### 4-1-1. TAM-SAM-SOM 분석

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │                     TAM: 5조원                          │   │
│   │              (국내 피트니스 시장 전체)                   │   │
│   │                                                         │   │
│   │   ┌─────────────────────────────────────────────────┐   │   │
│   │   │              SAM: 1.5조원                       │   │   │

```
┌─────────────┐     회원 초대      ┌─────────────┐
│  트레이너    │ ─────────────────→ │    회원     │

**예상 바이럴 계수 (K-factor):**
- 트레이너 1명 → 평균 회원 15명 초대
- 회원 15명 중 1명이 다른 센터에서 PAL 추천
- **K = 1.0 이상 목표** (자생적 성장 가능)

### 회원 유입 퍼널: 트레이너를 통한 자연 유입

PAL은 B2B2C 모델로 **고객획득비용(CAC) 0원**을 실현합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                      B2C 회원 유지 퍼널                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1단계: 설치 (100%)                                                 │
│  └─ 트레이너 통해 반강제 설치                                        │
│  └─ CAC(고객획득비용) = 0원                                         │
│                              ↓                                      │
│  2단계: 활성 사용 (95%)                                             │
│  └─ PT 기간 중 트레이너와 연동되어 자연스럽게 사용                   │
│  └─ 평균 3개월 사용 (PT 기간)                                       │
│                              ↓                                      │
│  3단계: PT 종료 후 유지 (목표 60%)                                  │
│  └─ 데이터 Lock-in + 습관 + 무료 기능                               │
│  └─ 삭제할 이유 없음                                                │
│                              ↓                                      │
│  4단계: 유료 전환 (목표 10%)                                        │
│  └─ AI 프리미엄 기능 필요성 느낀 회원                               │
│  └─ 혼자 운동 지속하는 회원                                         │
│                              ↓                                      │
│  5단계: 재구매/재등록 (목표 30%)                                    │
│  └─ PT 재등록 시 동일 앱 사용                                       │
│  └─ 새 트레이너도 PAL 사용 유도                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 3개월 이상 사용 시 발생하는 Lock-in 효과

#### Lock-in 요소별 분석

| Lock-in 유형 | 상세 내용 | 심리적 효과 |
|-------------|----------|------------|
| **데이터 축적** | 90일 체중 변화, 36회+ 운동 기록, 200장+ 식단 사진 | 앱 삭제 시 모든 기록 소실 → 전환 비용 |
| **습관 형성** | 매일 체중 기록, 식단 사진, 인사이트 확인 습관 | 66일 이상 사용으로 자동화된 행동 |
| **성취감** | 변화 그래프, AI 칭찬, 목표 달성 배지 | 정서적 연결, 앱에 대한 애착 |
| **사회적 연결** | 트레이너와의 관계 유지, 기록 공유 | PT 종료 후에도 연락 유지 |

**습관 형성 연구 근거:**

| 연구 | 기간 | 출처 |
|------|------|------|
| 적응 기간 | 21일 | Maxwell Maltz, Psycho-Cybernetics (1960) |
| **습관 형성 평균** | **66일** | Phillippa Lally, UCL, European Journal of Social Psychology (2009) |
| 개인차 범위 | 18~254일 | 같은 연구 |

> **핵심 포인트:**
> - PT 평균 기간 3개월 (약 90일) > 습관 형성 기간 66일
> - **PT 종료 시점에 이미 앱 사용이 습관화됨**

## 4-8. 결론: 성장전략 핵심 메시지

### 핵심 경쟁력: 데이터 축적 + AI 인사이트

> **"GPT를 직접 쓰면 되지, 왜 PAL을 써야 하나요?"**
>
> GPT는 단발성 질문에 답하는 도구입니다.
> PAL은 수개월치 데이터를 축적하고, 그 데이터를 기반으로 예측과 인사이트를 제공합니다.
>
> **트레이너가 회원 20명의 6개월치 데이터를 매번 GPT에 입력하는 것은 비현실적입니다.**
> PAL은 자동으로 데이터를 수집하고, 정리하고, 분석하고, 예측까지 해줍니다.

### 시장 기회

> **"1.5조원 PT 시장에서 '데이터 기반 AI 관리' 솔루션은 PAL이 유일합니다.
