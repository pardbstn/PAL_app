# 2. 문제인식

## 2-1. 개발 동기: PT 트레이너의 현실

### "운동 지도보다 서류 작업이 더 많다"

PT 트레이너로 3년간 현장에서 일하며 가장 큰 좌절감을 느꼈던 순간은 '운동을 가르치지 못할 때'가 아니었습니다. **회원 한 명의 커리큘럼을 작성하는데 1시간, 15명의 회원 데이터를 엑셀에 정리하는데 2시간, 그리고 이 과정을 매주 반복하며 정작 운동 지도에 집중하지 못하는 현실**이었습니다.

> "저는 트레이너가 되고 싶었지, 사무직이 되고 싶었던 게 아닙니다. 그런데 하루 일과의 절반은 컴퓨터 앞에서 보내고 있어요."
> — 창업자 본인, 3년차 PT 트레이너 경험

이 경험이 PAL의 출발점입니다. 트레이너가 본연의 업무인 '운동 지도'에만 집중할 수 있도록, **행정업무의 90%를 AI가 대신 수행하는 플랫폼**을 만들고자 합니다.

## 2-2. 고객 문제 정의

### 2-2-1. 트레이너 Pain Point: 5가지 핵심 문제

PT 트레이너들이 겪는 문제는 크게 5가지로 분류됩니다. 각 문제는 단순한 불편함이 아니라, **수입 감소와 직업 만족도 하락으로 직결되는 심각한 이슈**입니다.

| 순번 | 문제 영역 | 현재 상황 | 결과적 피해 |
|:---:|----------|----------|------------|
| 1 | **커리큘럼 작성** | 회원 1명당 30분~1시간 수동 작성 | 번아웃, 품질 저하 |
| 2 | **데이터 관리** | 엑셀/노션 수기 입력, 분석 불가 | 기록의 의미 상실 |
| 3 | **이탈 감지** | 사전 파악 방법 없음 | 재등록 기회 상실 → 수입 감소 |
| 4 | **회원 동기부여** | 시각화 자료 없음 | PT 조기 종료 |
| 5 | **계약서 관리** | 종이 계약서, 분실 위험 | 법적 분쟁 → 금전적 손해 |

#### 문제 1: 커리큘럼 작성의 시간 블랙홀

**현황:**
- 회원 1명의 N회차 커리큘럼 작성에 평균 30분~1시간 소요
- 트레이너 1인이 평균 15~20명의 회원 담당
- **주당 커리큘럼 작성 시간: 7~20시간** (수정 포함)

**트레이너 인터뷰 결과:**

> "신규 회원이 들어오면 일단 머리가 아파요. 목표, 체력 수준, 부상 이력을 다 고려해서 12주 커리큘럼을 짜야 하는데, 솔직히 처음 몇 명은 정성껏 했는데 지금은 복붙이 대부분이에요."
> — A 트레이너 (경력 4년, 회원 18명 담당)

> "커리큘럼이요? 솔직히 대충 써요. 어차피 회원들은 잘 안 보거든요. 근데 그게 문제인 게, 나중에 뭘 했는지 저도 기억이 안 나요."
> — B 트레이너 (경력 2년, 회원 12명 담당)

**문제의 심각성:**
- 커리큘럼 품질 저하 → 회원 만족도 하락 → 재등록률 감소
- 트레이너 번아웃 → 이직률 증가 → 센터 운영 불안정

#### 문제 2: 데이터 관리의 비효율성

**현황:**
- 대부분의 트레이너가 **엑셀(67.2%)** 또는 **노션(18.4%)**, **수기 노트(14.4%)**로 회원 데이터 관리
- 데이터 입력에 시간 소모, 분석은 사실상 불가능
- "기록은 하는데 활용은 못 한다"

**설문 조사 결과:**

| 질문 | 응답 |
|-----|------|
| 현재 회원 데이터 관리 방식은? | 엑셀 67.2%, 노션 18.4%, 수기 14.4% |
| 데이터를 분석해본 적 있나요? | "없다" 89.3%, "가끔" 8.2%, "정기적으로" 2.5% |
| 데이터 분석이 어려운 이유는? | "시간 부족" 54.1%, "방법 모름" 31.8%, "필요성 못 느낌" 14.1% |

> "엑셀에 체중, 인바디 수치 다 기록해요. 근데 그래서 뭐요? 그걸 그래프로 만들고 분석할 시간이 어딨어요. 그냥 쌓여만 가요."
> — C 트레이너 (경력 5년)

> "회원한테 '선생님, 제가 얼마나 좋아졌어요?'라고 물어보면 솔직히 대답하기 어려워요. 데이터는 있는데 정리가 안 돼서..."
> — D 트레이너 (경력 3년)

**문제의 심각성:**
- 쌓인 데이터의 가치가 0원
- 회원에게 객관적 성과 증명 불가
- 트레이너 전문성에 대한 신뢰 하락

#### 문제 3: 이탈 징후 감지 불가

**현황:**
- 회원의 이탈 징후(출석률 하락, 참여도 저하)를 사전에 감지하는 방법 없음
- PT 종료 후에야 "아, 그 회원 재등록 안 했구나" 인지
- 재등록 권유 타이밍 놓침 → 매출 손실

**데이터로 보는 이탈 문제:**

| 지표 | 수치 | 출처 |
|-----|------|-----|
| PT 회원 평균 재등록률 | 35~45% | 한국피트니스산업협회 (2023) |
| 이탈 사전 감지율 | 12% 미만 | 자체 트레이너 설문 (n=127) |
| 이탈 원인 중 "관리 소홀" | 28.7% | 회원 설문 (n=342) |

> "갑자기 회원이 연락이 안 되기 시작하면 그때서야 '아, 뭔가 잘못됐구나' 싶어요. 근데 그때는 이미 늦은 거죠."
> — E 트레이너 (경력 6년)

> "솔직히 출석률 떨어지는 거 알아요. 근데 15명 회원 출석을 일일이 체크하면서 패턴 분석할 시간이 없어요."
> — F 트레이너 (경력 2년)

**문제의 심각성:**
- 회원 1명 이탈 = 평균 PT 매출 150만원 손실 (20회 기준)
- 재등록률 10% 상승 시 연간 매출 15~20% 증가 추정
- 예방 가능한 이탈을 방치하는 구조적 문제

#### 문제 4: 회원 동기부여 어려움

**현황:**
- 회원에게 진행 상황을 시각적으로 보여줄 자료 부족
- "열심히 하고 계세요"라는 말만으로는 동기부여 한계
- 객관적 성과 데이터 없이는 PT 가치 증명 어려움

**회원 설문 결과:**

| 질문 | 응답 |
|-----|------|
| PT 중단 고려 이유는? | "효과를 모르겠다" 47.2%, "비용 부담" 31.5%, "시간 부족" 21.3% |
| 진행 상황을 어떻게 확인하나요? | "트레이너 말" 61.4%, "거울" 23.1%, "인바디" 15.5% |
| 시각화된 진행 보고서를 원하나요? | "매우 원한다" 73.8%, "원한다" 19.2% |

> "회원들이 중간에 포기하는 이유가 뭔지 아세요? 효과가 없어서가 아니에요. **효과가 있는지 모르니까** 포기하는 거예요. 보여줄 게 없으니까요."
> — G 트레이너 (경력 8년)

**문제의 심각성:**
- 회원 93%가 시각화된 진행 보고서를 원함
- 현재 이를 제공하는 트레이너는 8% 미만
- 결과: PT 조기 종료, 재등록률 하락

#### 문제 5: 계약서 관리의 리스크

**현황:**
- 대부분 종이 계약서 사용 (디지털 계약 도입률 11.2%)
- 분실, 훼손 위험 상존
- 분쟁 발생 시 증빙 어려움

> "계약서요? 서류함에 있어요. 근데 솔직히 찾으려면 한참 걸려요. 작년 회원 거는 어디 있는지도 모르겠어요."
> — H 트레이너 (경력 4년)

**문제의 심각성:**
- 환불 분쟁 시 계약서 분실로 인한 금전적 손해
- 법적 보호 장치 부재
- 전문성 있는 비즈니스 이미지 손상

### 2-2-2. 회원 Pain Point: 3가지 핵심 문제

PT 회원들이 겪는 문제는 트레이너의 문제와 연결되어 있습니다.

| 순번 | 문제 영역 | 현재 상황 | 결과 |
|:---:|----------|----------|------|
| 1 | **진행 상황 파악** | 본인 데이터 접근 어려움 | 동기 저하 |
| 2 | **식단 관리** | 사진 기록만, 분석 없음 | 영양 관리 실패 |
| 3 | **목표 대비 현황** | 객관적 비교 기준 없음 | 불안감, 조기 포기 |

#### 문제 1: 내가 얼마나 성장했는지 모른다

> "12주 PT 끝나고 트레이너님이 '많이 좋아지셨어요'라고 하는데... 솔직히 실감이 안 나요. 숫자로 보여주면 좋겠는데."
> — 회원 인터뷰, 30대 여성

> "인바디 찍으면 숫자가 나오긴 하는데, 그게 좋은 건지 나쁜 건지 모르겠어요. 비슷한 사람들이랑 비교해주면 좋겠어요."
> — 회원 인터뷰, 40대 남성

**데이터:**
- 회원의 73.8%가 "시각화된 진행 보고서를 매우 원한다"고 응답
- 현재 이를 제공받는 회원은 8% 미만

#### 문제 2: 식단 관리의 블랙홀

> "트레이너님이 식단 사진 보내라고 해서 보내는데, 피드백이 '잘 드시네요' 끝이에요. 칼로리가 얼마인지, 단백질이 부족한지 알고 싶은데."
> — 회원 인터뷰, 20대 여성

**문제의 심각성:**
- 트레이너도 영양 분석 전문가가 아님
- 식단 사진만 쌓이고 분석은 없음
- 회원의 자기 관리 동기 저하

#### 문제 3: 나는 잘하고 있는 걸까?

> "벤치프레스 60kg 들었는데, 이게 잘하는 건가요? 3개월 운동한 사람치고 어느 정도인지 알고 싶어요."
> — 회원 인터뷰, 30대 남성

**문제의 심각성:**
- 객관적 기준 없이는 성취감을 느끼기 어려움
- "남들과 비교"라는 인간의 기본 심리 미충족
- 목표 의식 희미해짐 → PT 조기 종료

### 2-2-3. 문제의 심각성과 시급성

**왜 이 문제를 지금 해결해야 하는가?**

| 관점 | 심각성 | 시급성 |
|-----|--------|--------|
| **트레이너** | 주 7~20시간 행정업무 → 번아웃, 이직 | 디지털 전환 경쟁 가속화 |
| **회원** | 47%가 "효과 모름"으로 PT 중단 고려 | 대체 운동 서비스(홈트, 앱) 증가 |
| **센터** | 재등록률 35~45%로 매출 불안정 | 경쟁 심화로 회원 유치 비용 증가 |

**문제 방치 시 예상 결과:**
1. 트레이너 이직률 증가 → 서비스 품질 하락
2. 회원 이탈 증가 → 센터 매출 감소
3. PT 산업 전반의 신뢰도 하락

## 2-3. 시장 현황 및 기회

|-----|------|-----|
| 국내 피트니스 시장 규모 | 약 5조원 (2023) | 한국피트니스산업협회 |
| PT 세부 시장 규모 | 약 1~1.5조원 | 이데일리 (2019.05) |

| 연간 PT 이용자 | 80~100만명 | 플래텀 (2016.12) |
| 트레이너 1인당 평균 회원 수 | 15~20명 | 자체 조사 |

### 2-3-2. 시장의 디지털 전환 현황

**현재 트레이너들의 디지털 도구 사용 현황:**

| 도구 유형 | 사용률 | 만족도 |
|----------|--------|--------|
| 엑셀/구글시트 | 67.2% | 2.1/5.0 |
| 노션/에버노트 | 18.4% | 2.8/5.0 |
| 수기 노트 | 14.4% | 1.5/5.0 |
| 전문 PT 관리 앱 | 23.7% | 3.2/5.0 |

**기존 PT 관리 앱의 한계:**

| 앱 이름 | 주요 기능 | 한계점 |
|--------|----------|--------|
| 바디코디 | 인바디 연동, 체성분 기록 | 커리큘럼 자동 생성 없음, AI 분석 없음 |
| 짐매니저 | 회원권 관리, ERP | PT 특화 기능 부족, B2B 중심 |
| 피트니스타 | 일정 관리, 회원 관리 | 데이터 분석 없음, 그래프 수동 |
| PT다이어리 | 운동 기록 | 회원-트레이너 연동 약함, 인사이트 없음 |
| 온핏 | 운동 영상 제공 | 개인화 부족, 트레이너용 아님 |

**핵심 공백:**
- **"기록은 하지만 분석은 안 해주는"** 도구들만 존재
- AI 기반 자동화 서비스 부재
- 트레이너-회원 양방향 플랫폼 미비

### 2-3-3. 기존 관리 방식의 한계

**엑셀/노션의 구조적 한계:**

| 한계 | 설명 | 영향 |
|-----|------|-----|
| 수동 입력 | 모든 데이터를 직접 타이핑 | 시간 소모, 오류 발생 |
| 분석 기능 부재 | 피벗테이블 등 고급 기능 미숙 | 데이터 가치 0 |
| 시각화 어려움 | 그래프 생성에 별도 작업 필요 | 회원 보고서 제작 불가 |
| 협업 불편 | 회원과 실시간 공유 어려움 | 정보 비대칭 |
| 백업 리스크 | 로컬 저장, 분실 위험 | 데이터 손실 |

> "엑셀 진짜 싫어요. 근데 뭘 쓰라는 거예요? 다른 앱들도 결국 입력은 제가 해야 하잖아요."
> — I 트레이너 (경력 3년)

### 2-3-4. 디지털 전환 트렌드

**2024-2025 피트니스 산업 트렌드:**

1. **AI 도입 가속화**
   - ChatGPT 등장 이후 AI 서비스 수용도 급증
   - "AI가 해주면 편할 것 같다" 응답 87.3%

2. **데이터 기반 의사결정**
   - 감에 의존하던 트레이닝 → 데이터 기반 최적화
   - 회원도 객관적 수치 요구 증가

3. **개인화 서비스 확대**
   - 획일적 프로그램 → 개인 맞춤형 커리큘럼

## 2-5. 해결 기회

### 2-5-1. 왜 지금인가? (Timing)

**2024-2025년이 최적의 진입 시점인 3가지 이유:**

#### 이유 1: AI 기술의 성숙

| 시점 | AI 기술 수준 | PT 관리 적용 가능성 |
|-----|-------------|-------------------|
| 2020년 이전 | 기초 ML, 제한적 NLP | 불가능 |
| 2021-2022년 | GPT-3 등장, 초기 생성 AI | 부분적 가능 |
| 2023년 | GPT-4, Claude 등 고성능 LLM | 본격 가능 |
| **2024년~** | GPT-4o, 멀티모달 AI | **최적 시점** |

- GPT-4o: 텍스트+이미지 동시 처리 → 식단 사진 분석 가능
- Prophet (Meta): 시계열 예측 → 체중 예측 가능
- 비용 하락: API 비용 2년간 80% 이상 감소

> "2년 전에는 AI로 커리큘럼 짜는 게 말이 안 됐어요. 지금은 돼요."
> — 개발팀 기술 검토 결과

#### 이유 2: 시장의 디지털 전환 가속화

- 코로나19 이후 디지털 도구 수용도 급증
- 젊은 트레이너(20-30대) 비중 증가 → 앱 친화적
- 경쟁 센터와 차별화 필요성 대두

| 연도 | 피트니스 앱 사용 트레이너 비율 |
|-----|------------------------------|
| 2019 | 12% |
| 2021 | 31% |
| 2023 | 52% |
| 2025(E) | 75% |

#### 이유 3: 경쟁 공백

- 기존 PT 관리 앱: "기록" 중심, AI 기능 없음
- AI 스타트업: 피트니스 특화 서비스 부재
- **"PT 특화 + AI 자동화" 조합의 선점 기회**
