# PAL 개인정보처리방침

**시행일: 2025년 1월 27일**

PAL(이하 "서비스")은 이용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다. 본 개인정보처리방침은 서비스 이용 시 수집되는 개인정보의 항목, 수집 목적, 보관 기간 등을 안내합니다.

---

## 1. 수집하는 개인정보 항목

### 필수 수집 항목
| 항목 | 수집 시점 |
|------|----------|
| 이메일 주소 | 회원가입 시 |
| 비밀번호 (암호화 저장) | 회원가입 시 |
| 이름 | 회원가입 시 |
| 프로필 사진 | 서비스 이용 시 (선택) |

### 소셜 로그인 시 수집 항목
| 로그인 방식 | 수집 항목 |
|-------------|----------|
| 카카오 로그인 | 카카오 계정 ID, 이메일, 프로필 이미지 |
| 네이버 로그인 | 네이버 계정 ID, 이메일, 프로필 이미지 |
| Google 로그인 | Google 계정 ID, 이메일, 프로필 이미지 |
| Apple 로그인 | Apple 계정 ID, 이메일 |

### 서비스 이용 시 수집 항목
| 항목 | 설명 |
|------|------|
| 체성분 데이터 | 체중, 체지방률, 골격근량, BMI 등 |
| 운동 기록 | 커리큘럼, 세션 기록, 운동 수행 내역 |
| 목표 설정 | 목표 체중, 운동 목표 |
| 메모 및 피드백 | 트레이너-회원 간 소통 내용 |

### 자동 수집 항목
- 기기 정보 (OS 버전, 기기 모델)
- 앱 사용 로그 (접속 시간, 기능 사용 기록)
- 푸시 알림 토큰

## 4. 개인정보 제3자 제공

서비스는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우 예외로 합니다.

### 서비스 운영을 위한 위탁
| 수탁자 | 위탁 업무 | 제공 항목 |
|--------|----------|----------|
| Google Firebase | 인증, 데이터베이스, 클라우드 기능 | 이메일, 서비스 이용 데이터 |
| Kakao | 소셜 로그인 | 카카오 계정 연동 정보 |
| Naver | 소셜 로그인 | 네이버 계정 연동 정보 |
| Supabase | 파일 저장소 | 프로필 이미지 |

### 법령에 따른 제공
- 법령에 의해 요구되는 경우
- 수사 목적으로 법령에 정해진 절차에 따라 요청이 있는 경우

## 5. 개인정보 보호 조치

서비스는 이용자의 개인정보를 안전하게 보호하기 위해 다음과 같은 조치를 취합니다.

### 기술적 조치
- 비밀번호 암호화 저장 (Firebase Authentication)
- SSL/TLS 암호화 통신
- 데이터베이스 접근 권한 관리
- 정기적인 보안 점검

### 관리적 조치
- 개인정보 취급자 최소화
- 개인정보 보호 교육 실시

## 6. 이용자의 권리

이용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다.

| 권리 | 방법 |
|------|------|
| 개인정보 열람 요청 | 앱 내 설정 > 내 정보 |
| 개인정보 수정 | 앱 내 설정 > 프로필 수정 |
| 개인정보 삭제 (회원 탈퇴) | 앱 내 설정 > 계정 삭제 |
| 동의 철회 | 앱 내 설정 또는 이메일 요청 |

## 7. 만 14세 미만 아동의 개인정보

서비스는 만 14세 미만 아동의 회원가입을 받지 않습니다. 만 14세 미만 아동의 개인정보가 수집된 것을 알게 된 경우, 즉시 해당 정보를 삭제합니다.
