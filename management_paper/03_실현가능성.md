## 2-1-2. 회원 Lock-in 기능 구현 방안

### 데이터 축적에 의한 Lock-in 전략

PAL의 핵심 경쟁력은 **시간이 지날수록 전환 비용이 증가**하는 구조입니다. 회원이 3개월 이상 사용하면 축적된 데이터가 강력한 Lock-in 효과를 만듭니다.

#### PT 이용 기간 통계

| 지표 | 수치 | 출처 |
|------|------|------|
| **PT 패키지 평균 회차** | 20~30회 | 피트니스 업계 일반 |
| **주 2~3회 수업 시 소요 기간** | 2.5~4개월 | 계산 (30회 ÷ 주 2회 = 15주) |
| **트레이너 권장 최소 기간** | 3개월+ | 트레이너 커뮤니티 의견 |
| **습관 형성 평균 기간** | 66일 | Phillippa Lally, UCL (2009) |

> **핵심: PT 기간 ≥ 습관 형성 기간**
> - PT 평균 3개월 = 약 90일
> - 습관 형성 평균 66일
> → **PT 종료 시점에 이미 앱 사용이 습관화됨**

### Lock-in 4가지 메커니즘

#### 1. 데이터 축적에 의한 Lock-in

```
┌─────────────────────────────────────────────────────────────────────┐
│                   3개월간 축적되는 회원 데이터                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   체중 기록          운동 기록          식단 기록          인바디    │
│   ┌─────────┐       ┌─────────┐       ┌─────────┐       ┌───────┐  │
│   │ 90일    │       │ 36회+   │       │ 200장+  │       │ 3회+  │  │
│   │ 체중 변화│       │ 운동 이력│       │ 식단 사진│       │ 체성분│  │
│   └─────────┘       └─────────┘       └─────────┘       └───────┘  │
│                                                                     │
│   → 앱 삭제 시 모든 데이터 소실                                      │
│   → 다른 앱으로 데이터 이전 불가                                     │
│   → 전환 비용(Switching Cost) 발생                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**기술적 구현:**

| 데이터 | 저장 위치 | 시각화 | Lock-in 효과 |
|--------|----------|--------|-------------|
