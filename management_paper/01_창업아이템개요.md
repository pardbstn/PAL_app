# 1. 창업아이템 개요(Business Overview)

## 1-1. 아이템 개요

### 아이템명

**PAL (팔)** - Progress, Analyze, Level-up

### 핵심 가치 (One-liner)

> "회원 등록만 하면 끝. 커리큘럼, 그래프, 분석까지 AI가 다 해드립니다."

### 서비스 정의

PAL은 **PT 트레이너의 행정업무 자동화**를 통해 **PT 지도에만 집중**할 수 있게 하는 **AI 기반 회원관리 플랫폼**입니다.

| 구분 | 내용 |
|------|------|
| **서비스명** | PAL (팔) - Progress, Analyze, Level-up |
| **슬로건** | "기록하고, 분석하고, 성장하다." |
| **핵심 타겟** | PT 트레이너 (B2B2C) |
| **플랫폼** | 모바일 앱 (Android/iOS) + 웹 대시보드 |
| **비즈니스 모델** | Freemium SaaS (월 0원~3만원) |

### 개발 동기 및 배경

#### 트레이너의 현실: "운동 지도보다 서류 작업이 더 많다"

PT 트레이너의 본업은 '운동 지도'입니다. 하지만 현실은 다릅니다.

- 국내 PT 트레이너 **2.5만명**이 연간 **80~100만명** 회원을 관리
- 회원 1명당 커리큘럼 작성에 **30분~1시간** 소요
- 주당 **7~20시간**을 행정업무에 투입
- 대부분 **엑셀, 노션, 종이**로 수동 처리

#### 트레이너가 겪는 5가지 핵심 문제

| 문제 | 현황 | 결과 |
|------|------|------|
| **커리큘럼 작성** | 수동 작성 30분~1시간/회원 | 트레이너 번아웃, 본업 집중 불가 |
| **데이터 관리** | 엑셀/노션에 수기 입력 | 기록만 하고 활용 못함, 의미 상실 |
| **이탈 감지** | 파악 방법 없음 | 회원 이탈 → 트레이너 수입 감소 |
| **회원 동기부여** | 변화 시각화 자료 없음 | PT 효과 체감 어려움 → 조기 종료 |
| **계약서 관리** | 종이 계약서 보관 | 분실 위험, 법적 분쟁 가능성 |

#### 시장의 공백

기존 PT 관리 앱들은 **'기록'만 가능**합니다. 트레이너가 진정으로 원하는 **'분석 + 예측 + 자동화'** 영역은 **완전히 비어 있습니다**.

**PAL은 이 공백을 AI로 채웁니다.**

### 핵심 경쟁력 3가지

#### 1. AI 자동화 (시간 혁신)

| 기존 방식 | PAL |
|-----------|-----|
| 커리큘럼 작성: **30분~1시간** | **30초 이내** 자동 생성 |
| 그래프 제작: **15분~30분** | **즉시** 자동 생성 |
| 성과 분석: **분석 불가** | **실시간** AI 인사이트 |

**트레이너 행정업무 90% 감소** → PT 지도에만 집중 가능

#### 2. 데이터 네트워크 효과 (경쟁 진입장벽)

```
사용자 증가 → 데이터 축적 → AI 예측 정확도 향상 → 서비스 가치 상승 → 더 많은 사용자
```

- 회원 데이터 + 템플릿 축적 → 타 서비스 전환 시 **데이터 손실**
- 누적 데이터 기반 AI 학습 → **후발주자 진입장벽**

#### 3. 역할 재정의 (Win-Win 구조)

| 기존 | PAL |
|------|-----|
| **회원**: 수동적 수강생 | **회원**: 데이터 입력 주체 + AI 동기부여 수혜자 |
| **트레이너**: 기록 + PT 모두 담당 | **트레이너**: PT 지도에만 집중 |

> **"트레이너는 PT만, 나머지는 AI와 회원이"**

### 기술적 차별성

PAL의 4가지 핵심 AI 기능은 기존 PT 관리 시장에서 **전무한 영역**입니다.

#### (1) AI 커리큘럼 자동 생성

| 항목 | 내용 |
|------|------|
| **입력** | 회원 목표(다이어트/벌크업), 운동 경력(초보/중급/고급), 부상 이력 |
| **출력** | 4~8회차 운동 커리큘럼 초안 (종목, 세트, 반복, 중량 포함) |
| **기술** | OpenAI GPT-4o + 도메인 특화 프롬프트 |
| **성능** | 30초 이내 생성, 만족도 목표 85% 이상 |
| **특징** | Rolling Plan - 실제 수행 데이터 기반 다음 회차 자동 재조정 |

**Before vs After:**
- 기존: 회원 1명 커리큘럼 작성에 **30분~1시간**
- PAL: **30초 이내** 자동 생성 → 트레이너가 수정/확정만

#### (2) AI 체중/성과 예측 (Predictive Analytics)

| 항목 | 내용 |
|------|------|
| **체중 예측** | N주 후 예상 체중, 체지방률, 골격근량 |
| **운동 성과 예측** | 1RM 도달 예상일 (회원별 성장 속도 반영) |
| **목표 달성 예측** | 목표 체중/체형 도달 예상일 |
| **기술** | Prophet (Meta 시계열 예측) + Python Cloud Function |
| **성능** | 최소 4주 데이터 기반, 신뢰도 80% 이상 |
| **피드백 루프** | 예측 오차 → 모델 재학습 → 정밀도 향상 |

**예시 출력:**
> "현재 속도 유지 시 4주 후 72kg 예상 (신뢰구간 70~74kg)"

#### (3) AI 인사이트 자동 생성

**트레이너용 인사이트:**
- "김민수 회원이 2주간 **출석률 40% 하락**했습니다. 연락이 필요해 보입니다."
- "PT 종료 임박 회원 3명 중 **이수진 회원의 재등록 가능성이 높습니다**."
- "하체 중심 커리큘럼의 체지방 감량 속도가 상체 중심보다 **1.3배 빠릅니다**."

**회원용 인사이트:**
- "이번 주 단백질 섭취량이 목표보다 하루 평균 **25g 부족**해요."
- "스쿼트 중량이 4주 전보다 **15kg 늘었어요**! 다음 달에 100kg 도전 가능해요."
- "벌써 12회차 완료! 처음 왔을 때보다 **체지방 4.1kg 감소, 골격근 1.2kg 증가**했어요."

**비교 인사이트 (익명화 데이터 기반):**
- "회원님의 이번 달 출석률은 85%로, 비슷한 목표를 가진 회원들 **상위 20%**입니다."
- "벤치프레스 60kg은 운동 경력 3개월 기준 **상위 35%** 수준이에요."

| 항목 | 내용 |
|------|------|
| **기술** | GPT-4o + 행동 패턴 분석 알고리즘 |
| **트리거** | 출석률 하락, PT 종료 임박, 성과 달성 등 이벤트 기반 |
| **빈도** | 주 1회 자동 생성 + 이벤트 발생 시 즉시 알림 |

#### (4) 식단 Vision AI 분석

| 항목 | 내용 |
|------|------|
| **입력** | 음식 사진 1장 |
| **출력** | 음식명, 예상 칼로리, 탄수화물/단백질/지방 g수 |
| **기술** | OpenAI GPT-4o Vision |
| **성능** | 한국 음식 인식률 80% 이상 |
| **추가 기능** | 주간 영양소 부족분 알림, 식단 개선 제안 |

**예시:**
> 📸 김치찌개 사진 업로드 → "김치찌개 1인분: 약 350kcal, 탄수화물 25g, 단백질 18g, 지방 12g"

### PAL의 가치 제안

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  "트레이너는 PT만, 나머지는 AI와 회원이"                            │
│                                                                 │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐         │
│  │    회원     │    │     AI     │    │  트레이너   │         │
│  │  데이터     │ →  │  분석/예측  │ →  │  PT 지도    │         │
│  │  직접 입력   │    │  자동 수행  │    │  에만 집중  │         │
│  └─────────────┘    └─────────────┘    └─────────────┘         │
│                                                                 │
│  회원이 데이터를 직접 입력하면, AI가 분석/예측/인사이트를 자동 생성.  │
│  트레이너는 행정업무에서 해방되어 PT 지도에만 집중하고,             │
│  회원은 자신의 변화를 직접 확인하며 동기를 얻는 Win-Win 구조.       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
