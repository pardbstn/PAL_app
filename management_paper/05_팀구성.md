### 개발 포트폴리오: 피트니스 앱 3종 + PAL

대표자는 PAL 이전에 **피트니스 도메인 앱 3종**을 직접 기획·개발하며 시장 검증과 기술 역량을 축적하였음

| 순서 | 앱 | 검증 내용 | 검증 규모 | 학습 성과 |
|:----:|------|----------|:---------:|----------|
| 1 | 러닝앱 | 운동 기록 습관화 UX 설계 학습 | - | 사용자 경험 설계 역량 |
| 2 | 운동 리워드앱 | 동기부여 메커니즘 실험 | - | 게이미피케이션 설계 역량 |
| 3 | AI PT앱 | PT 회원 대상 인터뷰 → 트레이너 구조적 문제 발견 | 회원 23명, 트레이너 12명 | 도메인 Pain Point 도출 |
| **4** | **PAL (현재)** | **트레이너 회원관리 자동화 플랫폼** | **베타 50명 진행 중** | **MVP 개발 완료, 앱스토어 출시** |

3번째 앱(AI PT) 개발 과정에서 트레이너의 구조적 문제를 발견하고, 이를 해결하기 위해 4번째 앱(PAL)을 개발하게 된 **검증 기반 창업 과정**임

### 기술 역량 상세

| 기술 영역 | 상세 | PAL 적용 |
|----------|------|----------|
| **Flutter** | Dart 기반 크로스플랫폼 앱 개발, Riverpod 상태관리, go_router 라우팅 | 트레이너/회원 통합 모바일 앱 전체 개발 |
| **Firebase** | Auth(인증), Firestore(DB), Cloud Functions(서버), Hosting(웹) | 서버리스 백엔드 전체 구축 |
| **AI 연동** | OpenAI GPT-4o API, Vision API, 프롬프트 엔지니어링 | AI 커리큘럼 생성, 식단 분석, 인사이트 6종 기능 구현 |
| **Python** | 데이터 분석, 통계 모델링 | 체성분 예측 로컬 알고리즘 개발 (API 비용 0원) |
| **배포** | iOS App Store, Google Play Store 심사 대응 | iOS/Android 동시 출시 완료 |

### 협업 상세

**OO PT센터 협력**
- 역할: 베타 테스트 현장으로 활용, 소속 트레이너 3~5명이 PAL 실사용 후 피드백 제공
- 기대 효과: 현장 기반 기능 검증, 트레이너 관점 UX 개선, 초기 사용자 확보 채널
- MOU 내용: 베타 테스트 협력, 트레이너 대상 PAL 홍보 협조, 추후 센터 단위 도입 검토

**UI/UX 디자인 외주**
- 역할: 현재 대표자가 직접 설계한 UI를 전문 디자이너가 리디자인
- 범위: 트레이너 앱 주요 화면 20종, 회원 앱 주요 화면 15종, 웹 관리자 페이지 10종
- 외주 비용: 약 200만 원 (1단계 사업비 외주용역비에서 집행)
